# CodeCelix - AI Business Automation Website

## Quick Overview
Modern React website showcasing AI business automation solutions with responsive design and performance optimizations.

## Tech Stack
- **Frontend**: React 19 + Vite + Tailwind CSS 4.x
- **Routing**: React Router DOM
- **Build**: Rolldown (Vite alternative)
- **Styling**: Tailwind CSS with custom configurations

## Key Features
- Responsive design across all devices
- Performance-optimized with lazy loading
- Code splitting for faster load times
- Hardware-accelerated animations
- Built-in performance monitoring

## Quick Start
```bash
npm install
npm run dev
```
Runs on `http://localhost:3001/`

## Project Structure
```
src/
├── components/     # Reusable UI components
├── pages/         # Route components (Home, Solutions, Industries, Pricing)
├── layouts/       # Navbar, Footer
├── assets/        # Images, static files
└── utils/         # Performance helpers
```
